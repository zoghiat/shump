Bonus Features Implemented

The bonus section of the project has been implemented successfully.

We added a complete Power-Up System as one of the bonus parts of the project. The game includes four different power-ups: Triple Shot, Shield, Health Pack, and Fire Rate Booster.

Triple Shot allows the player to fire three bullets at the same time. Shield gives the player temporary protection against enemy bullets and collisions. Health Pack restores the player’s health. Fire Rate Booster increases the player’s shooting speed for a limited time.

These power-ups can appear after defeating enemies. When the player collides with a power-up, its effect is applied to the player during the game.

In addition to the power-up system, the project also includes multiple enemy types with different behaviors, wave-based progression with increasing difficulty, HUD information such as score, coins, health, wave number, and elapsed game time, sound effects, background music, and SQLite-based storage for coins, high score, purchased items, and equipped items.

Overall, the bonus part of the project has been completed and integrated into the main gameplay system.
